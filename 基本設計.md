# obsidian-plugin-imagepaste

## 概要

obsidian のプラグイン作成。  
alt + V で md に画像を貼り付けた際、あらかじめ設定した命名ルールで画像が添付される。

## 背景

markdown のメモツールに obsidian を導入したい。  
しかし、画像の貼り付けが独自パス記法なので厄介。

vs code の拡張機能で paste images が素晴らしいため、obsidian のプラグインで実現したい。
最終的には公開もできればなおよし。

### paste image 使用例

```json
{
  // pasteImage
  "pasteImage.defaultName": "[image]YYMMDD_HHmmss",
  "pasteImage.path": "${currentFileDir}/images",
  "pasteImage.insertPattern": "<img src='./images/${imageFileName}' alt='image' style='width: 600px; border: 1px solid black;'>",
  "pasteImage.namePrefix": "${currentFileNameWithoutExt}_"
}
```

## 仕様書

### フロントエンド

obsidian の設定画面から、下記の 4 つの入力欄が表示されること。

| 項目                   | 値                                                                                                 |
| ---------------------- | -------------------------------------------------------------------------------------------------- |
| 画像保存先フォルダパス | ${currentFileDir}/images                                                                           |
| 画像命名ルール         | [image]YYMMDD_HHmmss                                                                               |
| 画像挿入パターン       | `<img src='./images/${imageFileName}' alt='image' style='width: 600px; border: 1px solid black;'>` |
| 画像命名接頭辞         | ${currentFileNameWithoutExt}\_                                                                     |

### 動作

obsidian のメモ画面で alt + V を押下すると、

1. クリップボードの画像を取得
2. 画像保存先フォルダパスに画像を保存
3. 画像挿入パターンに従い、画像をメモに挿入
